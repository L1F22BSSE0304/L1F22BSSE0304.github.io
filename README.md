This portfolio is highly customizable. To make it your own, you will primarily edit the content within the index.html file.

Section

Key Content to Update

HTML Elements/Tips

Identity

Name, Profession, Titles

Update <title> tag and <span class="text-accent">Sarah Chen</span> in the Hero section.

Colors

Accent Color (Emerald)

Modify the tailwind.config script block in the <head> section. Change 'accent': '#10b981' to your desired hex code (e.g., #2563eb for blue).

About Me

Professional Summary

Update the <p> tags within the <section id="about"> block.

Skills

List of technologies

Modify the <p> content inside the skill cards in <section id="skills">.

Projects

Titles, Descriptions, Tech Tags, Links

Update the content and links (href="#") within the <section id="projects"> cards.

Contact/Footer

Social Media Links

Update the href attributes in the footer's <a> tags with your GitHub, LinkedIn, and email.
